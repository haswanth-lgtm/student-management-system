# Student Management System

## Overview

The Student Management System is a Python-based application that helps manage student records efficiently. It allows users to add, view, update, and delete student information using a simple menu-driven interface.

## Features

* Add Student Records
* View Student Details
* Update Student Information
* Delete Student Records
* SQLite Database Integration
* Easy-to-use Interface

## Technologies Used

* Python
* SQLite
